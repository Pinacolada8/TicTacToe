# TicTacToe
TicTacToe Game

## Installation

Download and install .Net Core 6 SDK [.Net Core Install instructions](https://dotnet.microsoft.com/en-us/download).
You can use any IDE you feel confortable editing, but Visual Studio is preferred.

## Executing
Execution:
````bash
$ dotnet run
````
